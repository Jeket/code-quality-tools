# `@strv/prettier-config`

> STRV's config for Prettier

## STRV Config

package.json

```json
{
  "prettier": {
    "extends": ["@strv/prettier-config"]
  }
}
```

.prettierrc or .prettierrc.json

```json
{
  "extends": ["@strv/prettier-config"]
}
```

prettier.config.js or .prettierrc.js

```js
module.exports = {
  extends: ['@strv/prettier-config']
}
```
